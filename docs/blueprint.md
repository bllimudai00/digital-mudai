# **App Name**: Aetherium Forge

## Core Features:

- Interactive Mining Canvas: A canvas that responds to user input (clicks, drags) with visual effects, simulating the extraction of virtual resources. Different user input methods will yield different visual results.
- Resource Accumulation Display: Dynamically display the quantity of accumulated virtual resources.
- Visual Tool Selector: Allow users to switch between different visual "mining tools," which change the style and nature of the visual effects on the canvas. A generative AI tool decides the nature of the generated mining results, so each "tool" will create results subtly but distinctly different from all the others.
- Mining Log: Track each mining session, by logging each time a mining event took place, and displaying all past events to the user.

## Style Guidelines:

- Background color: Deep space black (#03071E) to evoke a sci-fi theme.
- Primary color: Electric indigo (#6A4C93) for main interactive elements.
- Accent color: Neon pink (#F72585) to highlight active elements and virtual resources.
- Body and headline font: 'Space Grotesk', a sans-serif with a computerized feel.
- Use glowing, geometric icons for resources and tools, styled to resemble futuristic interfaces.
- A split layout with the interactive canvas taking up most of the screen, and a sidebar for the resource display and tool selector.
- Use smooth, energetic animations for the mining effects and resource accumulation to emphasize the gamified feel.
